## ğŸ¯ Overview

The CYNO AI Report Analysis system is a sophisticated medical document processing pipeline that:

1. **Extracts text** from medical reports (PDFs, images) using PaddleOCR
2. **Structures data** using a two-stage LLM pipeline (LLM-A â†’ LLM-B)
3. **Validates findings** to prevent hallucinations
4. **Stores results** persistently in the database
5. **Provides real-time status** tracking via jobId
